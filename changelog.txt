Version 1.1.0
Released: 2026-06-26

[NEW]

[IMPROVED]

=========================================

Version 1.2.0
Released: 2026-06-27

[NEW]
- Added StreamMatrix Pro for live streaming (RTMP/RTMPS) with Live Encode CBR and Smart Auto-Reconnect features.
- Added Ping-Pong Loop ULTIMATE with 5 dynamic looping modes (Classic, DJ Stutter, Boomerang, Heartbeat, and Slow-Mo).
- Added real-time clean progress indicators (time-only) for Ping-Pong Loop processing.
- Added platform support info for streaming (YouTube, Twitch, Facebook, TikTok, Kick, Restream, etc).

[IMPROVED]
- Updated matrix launcher menu interface to integrate the new ultimate modules.
- Enhanced time parsing system to support flexible formats (e.g., 45s, 5m30s, 1h15m30s).
- Implemented a "Press ENTER to return" loop to prevent the script from exiting immediately after a task finishes.

[CHANGED]
- Renamed the video downloader module to "FetchMatrix Pro" to clarify its function and separate it from the new StreamMatrix Pro.

=========================================

Version 1.3.0
Released: 2026-06-27

[SECURITY]

- Fixed a critical security vulnerability (Device-Lock Bypass) where users could access the main menu using someone else's registered license.
- Implemented strict device validation that actively cross-checks the current phone model with the data registered on the server.
- Added an automatic protection system that wipes the local license file and terminates the session if a mismatched license is detected.

[FIXED]

- Resolved the infinite loop glitch that triggered repeatedly when a valid license key was inserted but the device was not yet registered on the server.

[IMPROVED]

- Optimized the background validation code to make license checks lighter and faster during startup.
